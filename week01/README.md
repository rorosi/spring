## 1주차


+ Dependencies
  + Lombok
  + Spring Web
  + Spring Data JPA
  + H2 Database
  + MySQL Driver  
<br/>
  
+ Rest
  + 서버의 응답이 JSON 형식
  + HTML, CSS 등을 주고받을 때에는 Rest x
<br/>
  
+ Controller
  + 클라이언트의 요청(Request)을 전달받는 코드
  + `@RestController`: JSON만 반환하는 경우
  + `@GetMapping`: 브라우저에서 주소를 치는 행위를 GET 방식으로 정보를 요청한다고 말함
